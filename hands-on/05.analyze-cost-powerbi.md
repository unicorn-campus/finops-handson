### 비용 데이터 내보내기 설정   
비용 데이터를 생성한 Storage 계정의 Blob 저장소로 내보내는 설정    
  
- 비용 데이터 내보낼 스콥 선택 
  MCA는 관리그룹별 비용관리가 안되므로 구독을 선택합니다.   
  EA 사용 시에는 관리그룹을 지정하면 됩니다.   
  ![](images/2026-07-20-14-34-04.png)   

- '만들기' 클릭 후 모든 데이터 선택  
- 데이터 세트 > 내보내기 접두사는 'all'로 지정: 실제로는 조직명 사용    
- 이미지와 같이 선택    
  생성한 스토리지 계정 지정   
  컨테이너명과 디렉토리 명 지정   
  형식과 압축 유형은 반드시 Parquet과 Snappy로 함   

  ![](images/2026-07-20-14-37-05.png)   

- 데이터 내보내기 
  내보내기 설정 클릭 후 상단의 '선택한 날짜 내보내기'에서 날짜 지정. 최대 1달임      
  ![](images/2026-07-20-14-39-59.png)     

- 내보내기 설정 페이지 하단에 실행 기록에 '성공'이라고 나와야 함   
  ![](images/2026-07-20-14-40-43.png)
