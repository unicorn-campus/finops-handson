## 실습 Git 소스 다운로드   
- Window 사용자 
  - PowerShell 실행
    ![](images/2026-07-20-12-16-23.png)  
        
  - 아래 명령 수행 
    ```
    cd ~
    mkdir workspace
    cd workspace
    git clone https://github.com/unicorn-campus/finops-handson
    ``` 
    
- Mac 사용자 
  - 터미널 실행
  - 아래 명령 수행 
    ```
    mkdir -p ~/workspace && cd ~/workspace
    git clone https://github.com/unicorn-campus/finops-handson
    ```
