# 실습 사전 준비

## 최초 로그인
- Azure 포탈 접근
  https://portal.azure.com/

- 다른 계정 사용 클릭

- email과 초기 암호 입력
  ex) b2b32@ktaiacademy.onmicrosoft.com

- 암호 변경: 초기 암호를 본인 암호로 변경  

- 디바이스 등록
  - 'Microsoft Authenticator' 앱 설치
  - 화면에 나온 QR코드를 Authenticator 앱의 QR 찍기로 찍음
  - Authenticator에서 등록한 Device를 눌러 코드 입력

## 실습 Git 소스 다운로드   
- Window 사용자 
  - PowerShell 실행
  - 아래 명령 수행 
    ```
    cd ~
    mkdir workspace
    cd workspace
    git clone https://github.com/unicorn-campus/finops-handson
    ``` 
- Mac 사용자 
  - 터미널 실행
  - 아래 명령 수행 
    ```
    mkdir -p ~/workspace && cd ~/workspace
    git clone https://github.com/unicorn-campus/finops-handson
    ```
