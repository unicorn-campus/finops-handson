# 11. Commitment discounts (약정 할인)

CostManagementConnector 샘플 보고서의 **Commitment discounts** 페이지 해설임.
"이미 산 약정(RI/Savings Plan)을 잘 쓰고 있나?"를 보는 활용률(건전성) 점검 화면임 (뒤 12~15번의 "더 사야 하나?"와 대비).

## 상단 3대 지표 (여기서 제일 중요한 건 Utilization)

| 지표 | 값 | 의미 |
|---|---|---|
| **Effective cost** | **1,256.9** | 약정이 적용된 비용 (07번 PricingCategory "Committed 1,256.9"와 일치) |
| **Utilization (활용률)** | **100.0%** | 산 약정을 얼마나 쓰고 있나 — 핵심 건전성 지표 |
| **Commitment savings** | **1.04천** | 약정으로 아낀 금액 |

> **Utilization = 약정 건전성의 심장.** 100%면 이상적(낭비 0), 낮으면 안 쓰는 약정 = 돈 버림

## 차트
- 일자별 약정 사용량 ~33.4/일, 색상 = **Used(짙은 남색) / Unused(옅은 파랑)**
- 전부 짙은 색 → 매일 100% 소진 = 낭비 없음

## 인사이트 (읽는 법)
1. **전체 활용률 100% = 우수** — 산 약정을 다 쓰고 있어 낭비 없음. 이게 가장 좋은 상태
2. **유휴 Savings Plan 1건 (0%)** — 안 쓰이는 절감 플랜 존재. 비용은 0이지만 왜 0% 활용인지 조사 필요 (범위 미스매치·리소스 없음 등)
3. **최대 절감원 ProdDS1VMReservation(AKS) 381.71** — AKS 예약이 가장 큰 효자
4. **List=Contracted 전부 동일** — 협상 할인 부재 재확인 (07·09번과 일관)
5. **11번 vs 12~15번 역할 구분**
   - 11번 = "산 걸 잘 쓰나?" (Utilization/활용)
   - 12~15번 = "더 사면 얼마 아끼나?" (Coverage/권장)

## 약정 건전성 판단 기준
```
Utilization 높음(≈100%) → 건전. 필요시 커버리지 확대 검토
Utilization 낮음        → 낭비! 범위(scope) 조정 또는 교환/환불 검토
Unused 항목 존재        → 즉시 원인 조사
```

**한 줄 요약**: Commitment discounts는 "활용률 100%로 약정 건전, 단 유휴 Savings Plan 1건은 조사 필요,
AKS 예약이 최대 효자"를 보여주는 약정 활용률 점검 화면.
