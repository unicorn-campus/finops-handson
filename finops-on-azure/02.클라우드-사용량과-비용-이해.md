# 유닛 2. 클라우드 사용량과 비용 이해 (Understand cloud usage and cost)

> 원본: [Understand cloud usage and cost](https://learn.microsoft.com/en-us/training/modules/adopt-finops-on-azure/2-understand-cloud-usage-costs) · 약 32분(실습 포함)  
> 도메인 ①. 콘텐츠 출처: FinOps Domain "Understand Cloud Usage & Cost" by FinOps Foundation

---

클라우드 사용량·비용 이해는 효과적인 클라우드 관리·최적화의 출발점임. 이를 통해 리소스가  
어떻게 활용되는지 통찰하고, 추세를 식별하며, 성능·비용 효율을 최적화하는 의사결정 가능.  
**데이터 수집·할당·보고와 분석·이상 관리** 4개 역량이 핵심 역할을 함.

---

## 역량별 목표

| 역량 | 목표 |
|---|---|
| **데이터 수집(Data ingestion)** | 다양한 소스의 데이터를 수집·전송·정규화하여 분석 — 클라우드 인프라의 사용량·비용 이해에 초점 |
| **할당(Allocation)** | 계정·태그·레이블·메타데이터로 공유 클라우드 비용을 배분·분할하는 전략 정의 → 팀·프로젝트 단위 책임성 창출 |
| **보고와 분석(Reporting + analytics)** | 클라우드 데이터를 검토·쇼백(showback)하여 사용·지출 패턴 통찰, 비용 최적화 기회 식별, 정보 기반 재무 의사결정 |
| **이상 관리(Anomaly management)** | 예기치 않은 비용·사용량 이상을 적시에 탐지·식별·알림·관리하여 비용 효율적 클라우드 운영의 위험 경감 |

---

## 🎬 강의 영상 정리 — "Microsoft 도구로 사용량·비용 이해하기"

> 공식 강의 영상(약 3분)의 한국어 자막을 정제한 요약임.

### 1) 데이터 수집

클라우드 환경의 사용량·비용 데이터 수집부터 시작함. Azure에서는 다음을 활용 가능:

- **Azure Monitor** — 리소스 사용량 데이터
- **Microsoft Cost Management 및 청구(Billing)** — 비용 데이터
- 타사(third-party) 도구 데이터도 통합 가능
- **Azure Data Factory · Azure Logic Apps** — 데이터 수집·정규화를 자동화하여 정확하고 포괄적인 분석용 데이터 확보

### 2) 할당(Allocation)

조직 내에서 비용이 어떻게 할당되는지 설정함. 부서·프로젝트·리소스 전반의 정확한 비용 할당은  
재무 책임성·최적화에 매우 중요함.

- **Microsoft Cost Management · Azure Policy** — 대규모로 리소스에 태그 지정, 비용 할당 규칙 설정
- 지출 추적용 사용자 지정 보고서 작성으로 비용을 효율적으로 배분

### 3) 보고와 분석(Reporting + analytics)

토대가 마련되면 보고·분석으로 심화함.

- **Microsoft Fabric · Power BI · Cost Management** — 대화형 대시보드·보고서 작성
- 클라우드 지출 시각화, 비용 절감 기회 파악, 핵심 성과 지표(KPI) 추적
- 클라우드 담당자·비용 관리자와 공유 → 적합한 인력이 자원 활용 최적화·미래 지출 예측·비용 효율 향상

### 4) 이상 관리(Anomaly management)

예상치 못한 비용에 대응함. 이상 탐지·해결은 비용 관리 유지와 예산 초과 방지에 중요함.

- **Microsoft Cost Management의 이상 탐지(anomaly detection)** — 비정상 지출 패턴·예기치 않은 비용 증가·잠재적 청구 문제를 자동 식별
- 이상 확인 시 근본 원인 해결 또는 예산·예측 조정
