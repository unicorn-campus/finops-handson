# 유닛 8. 요약과 추가 학습

> 원본: [Summary](https://learn.microsoft.com/en-us/training/modules/adopt-finops-on-azure/8-summary) · 약 2분

---

## 요약

본 모듈을 마치며, FinOps Foundation의 **도메인·역량**과 조직이 모범사례를 도입하도록 돕는  
**Microsoft 리소스·솔루션**을 두루 살펴봤음. 또한 FinOps 실천을 시작·성숙시키기 위한  
**FinOps 이터레이션 수행 방법**을 학습했음.

이제 조직의 고유 요구에 맞춘 정보 기반 의사결정에 필요한 지식·통찰을 갖췄음. 본 과정을 넘어,  
이 리소스·솔루션을 적용하여 효율을 추동하고 비즈니스 성장을 가속하는 모범사례를 구현할 것을 권장함.

---

## 핵심 정리 (도메인 → Azure 도구)

| 도메인 | 역량 수 | 대표 Azure 도구 |
|---|---|---|
| ① 사용량·비용 이해 | 4 | Cost Management, Azure Monitor, Data Factory, Logic Apps, Fabric, Power BI |
| ② 비즈니스 가치 정량화 | 5 | Pricing/TCO Calculator, Azure Migrate, Cost Management 예측·예산, Advisor |
| ③ 사용량·비용 최적화 | 5 | Well-Architected Framework, Advisor, Monitor, Reservations, Savings Plans, Hybrid Benefit, Carbon Optimization |
| ④ FinOps 실천 관리 | 8 | FinOps 문서, Review Assessment, 인터랙티브 가이드, Azure Policy |

이터레이션: **범위 정의(3~5개 역량) → 목표 식별(지식·프로세스·지표·도입·자동화) → 실행 → 리뷰** 반복.
